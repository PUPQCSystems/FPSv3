Faculty Performance System

System Overview

The Faculty Performance System (FPS) is a web-based platform designed to streamline the identification of key area of performance and ranking process for faculty at PUPQC.

The system aims to:

  • Provide a centralized and efficient way for academic head to evaluate performance data based on key areas.
  • Offer chairs and administrators a clear view of regular faculty performance across various key areas.
  • Automate calculations to generate faculty rankings based on pre-defined criteria.
  • Promote transparency and fairness in the evaluation process.

Login

Office of Academic Affairs Chairman will log their exisiting institutional credentials upon system activation.

Administrators will have access with designated login credentials.

Access

The FPS is accessible through most web browsers on desktops, laptops, and tablets.

For optimal performance, we recommend using most common web browsers such Edge, Chrome, Firefox

Troubleshooting

In case of forgotten passwords, you can can contact the system administrator and follow the instructions to reset your password.

If you encounter unexpected system behavior, please refresh the page or log out and log back in.

Ensure you are using a supported web browser and have a stable internet connection.

Review the user guide for specific instructions on how to use FPS and report access.

If any system issue persists, please report it to the IT Helpdesk at PUPQC Helpdesk.

Data Security

Polytechinc University of the Philippines is committed to protecting the privacy and security of all faculty data stored within the FPS. All data is encrypted and access is restricted to authorized users only.

Versioning

The FPS may undergo periodic updates and improvements. The latest user documentation and training materials will be made available on the Office of the Academic Affairs.
